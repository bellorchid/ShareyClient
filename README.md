# ShareyClient
A little study message share client for Windows、Linux and MacOS platform

* **一个基于学习信息链接分享的客户端,适用于Windows、Linux和MacOS平台，用来收集上网遇到的有趣的网站和随手记的客户端**
* **技术采用nodejs，electron，reactjs，materialUI**
